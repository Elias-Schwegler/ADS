Übungs-Setup für Python im Modul "Algorithmen & Datenstrukturen (ADS)"
======================================================================


Der Setup für die Übungen ist derart, dass für das ganze Semester mit einem
einzigen Projekt in der Entwicklungsumgebung (Eclipse, IntelliJ, VSCode, etc.)
gearbeitet werden kann.

1. Auspacken von 'ADS_Uebungen_Python.zip'.

2. Das neue Verzeichnis 'ADS_Uebungen_Python' enthält die Verzeichnis-Struktur
   für alle Übungen, wobei die Ausgangslage der Übung01 bereits enthalten ist.

[
3. Wenn mit Eclipse gearbeitet wird:
   Sicherstellen, dass das PyDev-PlugIn in Eclipse installiert ist, z.B. mit:
   www.ics.uci.edu/~pattis/common/handouts/pythoneclipsejava/eclipsepython.html

4. Import des Projektes in Eclipse:
   - Menü: File > Import... > General > Existing Projects into Workspace
     - Select root directory: C:\...\ADS_Uebungen_Python
     - Button: Refresh
     - in 'Projects' sollte erscheinen:
       ADS_Uebungen_Python (C:\...\ADS_Uebungen_Python)
     - Finish
]
